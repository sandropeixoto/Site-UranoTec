# Urano Inteligência & Tecnologia

## Resumo do Projeto
A **Urano Inteligência & Tecnologia** atua na vanguarda da inovação digital, oferecendo soluções de tours virtuais 360º de alto impacto. O software foca em apresentar os espaços de empresas com tecnologia de ponta. Este projeto consiste na sua Landing Page oficial: um ambiente moderno, rápido e responsivo, focado em transmitir acessibilidade tecnológica, gerar impacto visual imersivo e atrair novos clientes corporativos buscando experiências inovadoras em 3D.

## Funcionalidades Principais

> **Apresentação em Vídeo Imersivo**  
> Player de vídeo modal integrado diretamente na landing page, projetado para iniciar automaticamente (Autoplay) sem redirecionar o usuário do escopo da plataforma ou abrir novas janelas.

> **Design Dinâmico Guiado por Scroll**  
> Interação aprimorada com o uso engenhoso de animações de entrada que acompanham a navegação do usuário, criando uma experiência elegante e menos estática.

> **Showcase de Portfólio Interativo**  
> Grid dinâmico e atrativo para apresentação dos parceiros e projetos anteriores da marca.

> **Totalmente Responsivo**  
> Interface programada e estruturada segundo a abordagem *Mobile-First*, garantindo layout excelente desde smartphones a telas ultrawide.

> **Pacotes e Mapeamento de Serviços**  
> Listagem interativa de propostas de valor focadas na redução de custos de deslocamento, marketing contínuo 24/7 e detalhamento tecnológico (Puck, Oberon e Titânia).

## Stack Técnica

| Tecnologia | Descrição / Implementação |
| :--- | :--- |
| **TypeScript** | Superset que garante robustez no código e proteção contra erros em tempo de desenvolvimento. |
| **React** | Biblioteca central para a criação de componentes padronizados focados em interfaces dinâmicas. |
| **Vite** | Ferramenta de build de ambiente de desenvolvimento moderna e extremamente rápida. |
| **Tailwind CSS**| Framework de uso intensivo neste projeto para um design e layout sofisticado de forma escalar. |
| **Motion** | Lib poderosa encarregada do sistema fluído de todas as animações (Fade, Scrim e Scales). |
| **Lucide React** | Biblioteca de ícones vetoriais modernos, limpos e acessíveis. |

## Instalação 

Siga os comandos abaixo na sua interface de terminal para testar o sistema localmente (é requerido ambiente Node.js LTS).

1. Clone o repositório para o seu ambiente local:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Acesse a raiz do diretório recém-clonado:
```bash
cd Site-UranoTec
```

3. Instale todas as bibliotecas e pacotes declarados nas dependências:
```bash
npm install
```

4. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

A aplicação agora deve estar disponível no navegador através do endereço genérico `http://localhost:3000`.
